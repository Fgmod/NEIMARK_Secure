# NEIMARK_Secure

Приложение "Neimark Secure": подробная интсрукция использования. 

1. Скачиваем архив с файлами на свой пк(желательно с Яндекс Диска), извлекаем в отдельную папку и запускаем файл NEIMARK_Secure.exe

2. Заранее на свой телефон скачиваем приложение Google Authenticator.

3. Запускаем приложение, открываем вкладку "Регистрация" и вводим нужные данные. 

4. На следующей вкладке появляется QR-код, который нужно отсканировать в приложении Google Authenticator(пункт 2). 
(В приложении нажимаем на плюс в правом нижнем углу, нажимаем на кнопку "Сканировать QR-код" и наводим камеру на QR в приложении NEIMARK Secure)

5. В приложении появляется наш аккаунт со специальным кодом, который обновляется каждые 30 секунд. Вводим этот специальный код в отведенное для этого поле в NEIMARK Secure и нажимаем кнопку "Подтвердить". (При нажатии на кнопку "Пропустить" двухфакторная аутентификация пройдена не будет).

6. После этого нас перебрасывает на вкладку со входом в аккаунт, вводим свои данные и подтверждаем в появившемся окне свой вход кодом, который также присутвует в приложении Google Authenticator. 

7. Подтверждаем вход и заходим в приложение, все готово! Теперь можно заполнить вкладку "Профиль", посмотреть вкладки "Кампус NEIMARK", "Карта", "Логи", а также выйти в самое начало.
